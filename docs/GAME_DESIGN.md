# NetBite Game Design

## Vision

NetBite is a mobile-first networking learning game.

It combines:

- Interactive lessons
- Small labs
- Visual simulations
- Quizzes
- Flashcards

The player should learn by interacting rather than reading.

## Chapter Structure

Every chapter contains:

- Introduction
- Interactive lesson
- Mini Lab
- Quiz
- Flashcards

---

## Lab Philosophy

Labs should be:

Small

Focused

Short

Teach one concept.

Never introduce multiple concepts.

---

## Learning Philosophy

Show

Don't tell.

Prefer interaction over explanation.

Every failure should explain:

- Why
- How to fix
- Concept involved

---

## Progression

Players unlock chapters sequentially.

Completing chapters unlocks:

- New devices
- New commands
- New labs
- New flashcards
