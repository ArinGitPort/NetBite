# NetBite Product References

This folder records product decisions that apply across chapters. It complements the chapter content and visual theme documents.

- `USABILITY.md` defines the mobile interaction and accessibility baseline.
- `ENGAGEMENT.md` defines learning-focused engagement patterns.
- `ROBUSTNESS.md` defines validation, persistence, testing, and recovery requirements.
- `IMPLEMENTATION_ORDER.md` records the recommended sequence for applying these ideas.

The central rule is simple: learner actions should produce clear network responses and useful explanations. Do not add decorative systems that compete with the concept being taught.
