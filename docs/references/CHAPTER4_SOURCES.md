# Chapter 4 Technical Sources

## Primary references

- [RFC 791: Internet Protocol](https://www.rfc-editor.org/rfc/rfc791) — 32-bit IPv4 addresses and network/host addressing concepts.
- [RFC 1918: Address Allocation for Private Internets](https://www.rfc-editor.org/rfc/rfc1918) — private IPv4 address blocks.
- [RFC 4632: CIDR Address Strategy](https://www.rfc-editor.org/rfc/rfc4632) — prefix-length notation and address aggregation.

## Content boundaries

NetBite validates static beginner host settings. DHCP, IPv6, NAT, address exhaustion, and public allocation policy are out of scope. Examples use documentation-only private networks.
