# Network Scope

NetBite teaches networking concepts.

NetBite does NOT simulate networking protocols.

---

Supported

✓ IPv4

✓ Static Routing

✓ Switching

✓ ARP

✓ ICMP

✓ VLAN

✓ DHCP (simplified)

✓ DNS (simplified)

---

Not Supported

✗ Dynamic Routing

✗ OSPF

✗ BGP

✗ MPLS

✗ NAT overload simulation

✗ TCP congestion

✗ Packet fragmentation

✗ Real sockets

✗ Timing simulation

✗ Collision domains

✗ Electrical layer simulation

---

If a concept can be validated through graph state and configuration state, prefer that approach.
